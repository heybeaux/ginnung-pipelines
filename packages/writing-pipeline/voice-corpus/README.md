# voice-corpus

`raw/` holds scraped Tumblr posts from `beaux-beaux.tumblr.com` (one file per post, original markup preserved). `examples/` holds the normalized corpus the voice-critic reads (one essay-shaped file per example, plain text or markdown). `fingerprint-v1.json` is generated by `src/voice/corpus/fingerprint.ts` from `examples/` and is the artifact the voice-critic compares drafts against. Files under `raw/.cache/` are git-ignored.
